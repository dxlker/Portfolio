# Portfolio Website - Tseten Lobsang

## Beschrijving

Dit project is een persoonlijke portfolio website waarin ik mezelf, mijn vaardigheden en mijn projecten presenteer. Het doel van de website is om bezoekers snel kennis te laten maken met wie ik ben als Software Developer en welke projecten ik heb gemaakt.

De website is ontwikkeld met HTML, CSS, JavaScript en TypeScript. Daarnaast is er aandacht besteed aan responsive design en toegankelijkheid zodat de website goed werkt op verschillende apparaten.

## Functionaliteiten

* Over mij venster
* CV venster met geïntegreerde PDF-weergave
* Projectenoverzicht
* Projectpagina
* Responsive design voor desktop en mobiel
* Toetsenbordnavigatie en focus states voor toegankelijkheid
